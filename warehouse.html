ISSUE INVENTORY

Use this section when inventory is issued to a technician, job, work order, or client.

WAREHOUSE INVENTORY ADJUSTMENT

Use this section only for positive, audit-preserving warehouse adjustments such as found inventory, missed cycle count items, restock found, or physical count corrections. This does not edit or delete original transactions.
Positive adjustments only. Serialized items are blocked in this workflow so serial-number tracking stays accurate. Use Receive Inventory for serialized items.
Part description will appear here after a part number is selected.
Adjustment status will appear here...
