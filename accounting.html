Transaction Report

Use the dropdown to choose the transaction report type. Manual adjustments and cycle count adjustments are separated clearly, issued serialized inventory includes the actual serial number when available, and the export includes both the raw transaction type and a friendly transaction label.
All Adjustments includes both Manual Adjustment and Cycle Count Adjustment. Transaction Reversals includes receive, issue, adjustment, and cycle count reversal activity.

Inventory Valuation Ledger

Select a date range to view a true inventory valuation ledger. This report uses the same valuation basis throughout: historical quantity multiplied by the current part unit cost, so Beginning Inventory plus activity ties exactly to Ending Inventory.
