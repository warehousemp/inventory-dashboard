Step 1: Export Count Sheet

Location: Main Warehouse
Whole Location Count includes all non-serialized stock with quantity on hand greater than zero and all on-hand serialized items currently assigned to the selected location.
Main Warehouse by Category uses the existing category workflow for Main Warehouse. Whole Location Count is for Van 3, Van 4, Van 6, and WARRANTY only. Count sheet CSV / print / PDF uses clean counting columns, including Expected Serial Number for serialized items. 
Select a category and load count sheet...

Step 2: Create Count Session

Creating a cycle count session creates count records for the selected mode. Main Warehouse by Category uses the selected category. Whole Location Count uses the selected Van/Warranty location and category WHOLE_LOCATION. This does not change inventory quantities.
Active Cycle Count ID: None 
Create a cycle count session after selecting a mode, or open a recount from the Recount Required Queue...
Continue / Enter Counts

Step 3: Enter Counts and Verify Serialized Items

A cycle count session can include both non-serialized quantity lines and serialized verification lines. Enter counted quantities for non-serialized parts. Verify serialized items by serial number using FOUND, MISSING, WRONG LOCATION, DAMAGED, or NEEDS REVIEW. Saving records the physical count and verification results. Step 4: Submit for Review

Mark Ready for Review submits the completed count for Admin review. The page will block submission until all quantity lines have counted quantities and all serialized lines have verification results. This does not update stock balances and does not create inventory transactions.
